 Telegram Chat Moderator Bot

Асинхронний Telegram-бот для автоматичної модерації груп. Розроблений для підтримки порядку в чатах, фільтрації спаму та вітання нових учасників.

 Функціонал

* Автоматичне вітання: Персоналізоване повідомлення для нових учасників із закликом прочитати правила.
* Фільтр ненормативної лексики: Миттєве видалення повідомлень із забороненими словами.
* Anti-Spam (посилання): Блокування повідомлень, що містять зовнішні посилання (`http`, `t.me`).
* Система покарань (Mute): Автоматичне обмеження прав користувача (read-only) на 1 годину за порушення.
* Адмін-інструменти: Команда /unmute (через Reply) дозволяє адміністраторам знімати обмеження.

 Технології

* Python 3.10+
* aiogram 3.x (Асинхронна бібліотека для Telegram API)
* AsyncIO
* Logging (Для відстеження подій)
